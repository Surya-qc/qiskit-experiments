##### Qiskit Experiments



Quantum computing simulations built with Qiskit and Qiskit Aer.

Each experiment follows the full stack: ideal circuit → noise model

→ transpilation → error mitigation.







##### Experiments



###### 1\. Mach-Zehnder Interferometer (MZI)



Simulation of quantum interference, phase control, and which-path detection.

→ \[mzi/](mzi/)





###### 2\. Bell Inequality and CHSH Test



Simulation reproducing Aspect's 1982 experiment. S = 2.804, violating

the classical bound of 2.0 and approaching the Tsirelson bound of 2.828.

→ \[bell-chsh/](bell-chsh/)
